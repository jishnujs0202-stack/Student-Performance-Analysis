# Student Performance Analysis

## Project Overview
This project analyzes student performance data using Python. The dataset is explored, cleaned, and visualized to understand factors affecting student grades.

## Objective
- Load and explore the dataset
- Perform data cleaning
- Analyze student performance
- Find relationships between study time and grades
- Compare performance based on gender
- Create visualizations using Matplotlib

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset used is **student-mat.csv**, which contains information about students' academic performance and related attributes.

## Project Steps

### 1. Import Libraries
Imported Pandas, NumPy, and Matplotlib for data analysis and visualization.

### 2. Load Dataset
Loaded the dataset using Pandas and examined its structure.

### 3. Data Exploration
- Checked dataset shape
- Viewed column information
- Inspected sample records

### 4. Data Cleaning
- Checked for missing values
- Checked for duplicate records

### 5. Data Analysis
Performed the following analyses:
- Average Final Grade (G3)
- Number of Students Scoring Above 15
- Correlation Between Study Time and Final Grade
- Average Grade by Gender

### 6. Data Visualization
Created:
- Histogram of Final Grades
- Scatter Plot of Study Time vs Final Grade
- Bar Chart of Average Grade by Gender
